# Public Launch Kit

## 项目一句话介绍

《Who Holds the Camera? 西方镜头中的中国》是一个 AI 辅助的中文线上档案展览，追问摄影、权力、来源透明度和“中国”视觉想象之间的关系。

## 100 字中文简介

这是一个关于摄影、权力与记忆的中文线上展览。项目从 Flashbak 等二级视觉线索出发，继续追溯档案馆、博物馆、摄影师项目与公共资料，把 24 件图像材料组织成 7 个展厅，并公开标注来源状态、版权不确定性和观看问题。

## 300 字中文简介

《Who Holds the Camera? 西方镜头中的中国》不是一个怀旧老照片网页，也不是 Flashbak 中文版。它把二级传播中的视觉线索作为入口，继续回到 Wellcome Collection、Library of Congress、University of Bristol Historical Photographs of China、The Met、M+、摄影师项目和画廊页面等更接近原始来源的位置。

页面被设计成线上展览：有入口、策展人导言、展厅地图、5 条参观路线、7 个 Gallery、24 件展品、Object Label、Looking Question、Archive Table 和 Source Index。每件展品都标注来源状态，例如 verified archive source、rights unclear、secondary source only。授权不明确的图像不被伪装成公共领域素材，而是作为外链线索和研究路径展示。

这个项目也是一次 AI 辅助数字策展实验：AI 可以帮助快速搭建结构、文案和界面，但来源判断、权利说明和不确定性的呈现必须被显式设计出来。

## 英文简介

Who Holds the Camera? is a Chinese digital exhibition about photography, power, memory, and the visual construction of "China" through Western-facing image trails. The project starts from secondary visual leads such as Flashbak, then traces objects back toward archives, museums, photographer projects, galleries, university collections, and public-domain repositories.

Instead of presenting historical photographs as a nostalgic image grid, the site treats each object as a source trail. It includes 7 galleries, 24 object labels, 5 exhibition routes, a filterable Archive Table, and a Source Index that makes rights status and research limits visible.

## GitHub Pinned Repo 描述

Chinese digital exhibition on photography, power, and source transparency. Built with Astro and deployed on Cloudflare Pages.

## 作品集描述

An AI-assisted digital exhibition and static site project that combines visual archive research, museum-style interface design, source status transparency, and Cloudflare Pages deployment. The work demonstrates how historical image materials can be presented as an online exhibition rather than a nostalgic blog post or image dump.

## 项目亮点

- 7 个 Gallery，从摄影之前的图像复制到 1982 年开放初期旅行影像。
- 24 件展品，每件都有 museum-style object label 和 source trail。
- 5 条参观路线，支持快速参观、完整参观、香港影像、海外华人和图像技术路径。
- Archive Table 支持按来源状态筛选。
- Source Index 按 Archive / Museum / Library、Photographer / Estate / Gallery、Secondary visual trail、Further reading 分组。
- 公开显示 rights unclear 和 secondary source only，不把不确定性藏起来。
- Astro + TypeScript + CSS 静态站点，部署到 Cloudflare Pages。
