# Who Holds the Camera? 西方镜头中的中国

Role:
AI-assisted curator / frontend builder / visual archive researcher

Tools:
Codex, Astro, TypeScript, Cloudflare Pages

Problem:
历史图像网页常常只做怀旧图集，来源不透明，也很少说明观看者位置。

Solution:
把视觉材料设计成线上展览，加入总览展、Gallery 单页、Object Label、Study Wall、Archive Drawer、Lightbox Previous / Next、Source Index、Archive Table、Looking Question 和来源状态 badge。

Impact:
- 9 个 Gallery
- 9 个 Gallery 单页
- 90 件展品
- Gallery 07 上海镜头
- Gallery 08 宣传画与未来想象
- 5 条参观路线
- Lightbox 近看模式与 Archive Drawer 展品抽屉
- 50 个 verified archive source
- 38 个 rights unclear
- 2 个 secondary source only
- Cloudflare Pages 公开部署
