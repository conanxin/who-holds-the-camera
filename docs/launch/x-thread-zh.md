# X 中文长帖

1/10
我做了一个中文线上展览：Who Holds the Camera? 西方镜头中的中国。

它关于摄影、权力、记忆，以及“西方镜头中的中国”是如何被图像、标题、档案和传播路径共同塑造出来的。

2/10
这个项目不是 Flashbak 中文版，也不是老照片合集。

Flashbak 这类页面在这里更像入口：它们提供视觉线索，但不应该成为最终来源。真正的问题是：图像从哪里来？谁命名？谁拥有说明权？

3/10
所以我把工作流设计成：

从二级视觉线索出发，继续追溯到档案馆、博物馆、大学项目、摄影师官网、estate、画廊和公共领域资料库。

能回到原始馆藏页的，就回到原始馆藏页。

4/10
页面最后不是一篇文章，而是一个 digital exhibition。

有 Exhibition Entrance、Curator's Statement、Gallery Map、5 条参观路线、9 个 Gallery 总览区、9 个 Gallery 单页、Reflection Room、Archive Table 和 Source Index。

首页现在是总览展，不再把所有展品平铺成一个长列表。

5/10
每件展品都做成 museum label：

标题、年份、作者、地点、媒介、credit、来源链接、馆藏/项目页、rights note、caption、curatorial note 和 tags。

这些字段本身就是展览的一部分。

6/10
我特别想把“来源状态”做成可见的界面，而不是藏在文末：

- verified archive source
- rights unclear
- secondary source only
- placeholder pending replacement

rights unclear 不会被删掉，也不会被假装成公共领域素材。

7/10
目前展览里有：

- 9 个 Gallery
- 9 个 Gallery 单页
- 90 件展品
- 5 条参观路线
- 5 条 thematic trails
- Lightbox 近看模式与 Archive Drawer 展品抽屉
- Gallery 07 上海镜头
- Gallery 08 宣传画与未来想象
- 50 个 verified archive source
- 38 个 rights unclear
- 2 个 secondary source only

8/10
这也是一次 AI 辅助数字策展实验。

AI 可以帮我快速搭建结构、整理数据模型、生成界面和初稿文案。但来源透明、权利判断和“不确定性如何显示”，必须被明确设计出来。

9/10
我最在意的问题其实是这个：

当镜头看向中国，它看见的是中国，还是观看者自己？

照片中的沉默部分，和可见部分同样重要。

10/10
